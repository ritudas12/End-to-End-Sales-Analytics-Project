# Product Sales Analytics
# 📊 Product Sales Analysis (Excel,SQL Server & Power BI)

##  Project Overview

This project focuses on analyzing product sales data using an end-to-end data analytics approach. The data was sourced from Excel files, processed and analyzed in SQL Server, and visualized through an interactive Power BI dashboard.

---

##  Objective

To analyze product sales data and generate actionable insights on revenue, sales trends, and product performance using SQL Server and Power BI.

---

##  Tools & Technologies

* Excel (Data source)
* SQL Server (Data cleaning, transformation, and analysis)
* Power BI (Dashboard and visualization)

---

##  Dataset Description

The dataset contains product sales information such as:

* Product details
* Sales quantity
* Revenue / sales amount
* Order or transaction data
* Date/time of sales

---

##  Project Workflow

### 1. Data Collection

* Imported multiple Excel files containing product sales data

### 2. Data Processing (SQL Server)

* Loaded data into SQL Server
* Cleaned and transformed data
* Performed analysis using SQL queries:

  * Total revenue calculation
  * Total quantity sold
  * Order/transaction analysis
  * Product-wise performance

### 3. Data Visualization (Power BI)

* Connected Power BI to SQL Server
* Created an interactive dashboard
* Built visuals for:

  * Revenue trends over time
  * Top-selling products
  * Sales distribution by category
  * Monthly/weekly sales trends

---

## Key Insights

* Identified top-performing products based on revenue and quantity
* Analyzed sales trends across different time periods
* Discovered peak sales periods
* Compared product/category performance

---

##  Project Files

* `product_sales.csv ,discount_data.csv,Product_data.csv` → Raw dataset
* `Poduct_sale.sql` → SQL queries used for analysis
* `Product_sale_dashboard.pbix file` → Power BI dashboard
